# نظام المصادقة - Daily Task Manager

## التحديثات المضافة

تم إضافة نظام مصادقة كامل للمشروع يتضمن:

### Backend (Python)
- إضافة endpoints جديدة:
  - `POST /api/register` - تسجيل مستخدم جديد
  - `POST /api/login` - تسجيل الدخول
  - `GET /api/users` - عرض المستخدمين
- قاعدة بيانات للمستخدمين: `backend/users.db.json`

### Frontend (React)
- صفحة تسجيل الدخول (Login)
- صفحة التسجيل (Register)
- عرض اسم المستخدم في الهيدر
- زر تسجيل الخروج
- حماية الصفحة الرئيسية (لا يمكن الوصول إليها بدون تسجيل دخول)

## كيفية الاستخدام

1. شغل Backend:
```bash
python backend/server.py
```

2. شغل Frontend:
```bash
npm run dev
```

3. افتح المتصفح على `http://localhost:5174`

4. سترى صفحة تسجيل الدخول أولاً
   - إذا لم يكن لديك حساب، اضغط "إنشاء حساب جديد"
   - أدخل اسم المستخدم وكلمة المرور
   - بعد التسجيل الناجح، ستنتقل تلقائياً للصفحة الرئيسية

5. بعد تسجيل الدخول:
   - سيظهر اسم المستخدم في أعلى الصفحة
   - يمكنك استخدام جميع ميزات إدارة المهام
   - لتسجيل الخروج، اضغط زر "تسجيل الخروج"

## ملاحظات
- البيانات محفوظة في localStorage
- عند تسجيل الخروج، يتم حذف بيانات الجلسة
- النظام بسيط وآمن للاستخدام المحلي
